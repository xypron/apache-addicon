apache-addicon
==============

This package provides additional icons for directory listings.

Usage
-----

AddIcon is used in the Apache configuration to associate an icon to a
file extension.
----
AddIcon /icons/debian.png .deb
AddIcon /icons/signature.png .gpg
----
